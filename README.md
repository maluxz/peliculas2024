# Películas 2024

Este proyecto se encuentra en desarrollo y es parte de la materia de Dispositivos Móviles II en el Tecnológico Nacional de México (TecNM) campus Delicias.

## Descripción

Películas 2024 es una aplicación móvil desarrollada en Flutter que proporciona información sobre películas en cartelera, detalles de películas y más. La aplicación utiliza la API de IMDb para obtener datos de películas.

## Estado del Proyecto

Este proyecto está en desarrollo activo como parte de la materia de Dispositivos Móviles II en el TecNM Delicias. Actualmente se están implementando las funcionalidades básicas y se están explorando mejoras adicionales.

## Requisitos

- Flutter SDK
- Dart SDK
- Android SDK / Xcode para emulación y pruebas

## Instalación

1. Clona el repositorio a tu máquina local.
2. Asegúrate de tener Flutter y Dart SDK instalados.
3. Ejecuta `flutter pub get` para instalar las dependencias.
4. Ejecuta la aplicación en tu emulador o dispositivo usando `flutter run`.
